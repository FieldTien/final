# ML2018SPRING FINAL

*python 3.6.3
*tensorflow 1.4
*keras 2.0.8
*pandas 0.22.0
*pandas 1.14.5
*xgboost 0.71
*librosa 0.6.0

1.視您的環境情況決定是否需要安裝套件

`<addr>`bash  hw5_train.sh <train.csv>  <train.csv>  <output>
Example
`<addr>`bash final_test.sh "train.csv" "train.csv" "bashtest.csv"
